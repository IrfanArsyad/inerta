## CSS/Styling Improvements

### 1. Custom Checkbox Styling
```css
/* More modern checkbox */
.form-check-input:checked {
  background-color: var(--vz-primary);
  border-color: var(--vz-primary);
  animation: checkPop 0.2s ease;
}

@keyframes checkPop {
  0% { transform: scale(0.8); }
  50% { transform: scale(1.1); }
  100% { transform: scale(1); }
}
```

### 2. Table Row Hover Effect
```css
.table tbody tr {
  transition: background-color 0.15s ease;
}

.table tbody tr:hover {
  background-color: rgba(var(--vz-primary-rgb), 0.05);
}

.table tbody tr.selected {
  background-color: rgba(var(--vz-primary-rgb), 0.1);
}
```

### 3. Card Hover Effect
```css
.card-hover {
  transition: transform 0.2s ease, box-shadow 0.2s ease;
}

.card-hover:hover {
  transform: translateY(-2px);
  box-shadow: 0 0.5rem 1rem rgba(0, 0, 0, 0.08);
}
```

### 4. Status Badge with Pulse
```css
.status-badge.online::before {
  content: '';
  width: 8px;
  height: 8px;
  border-radius: 50%;
  background: var(--vz-success);
  margin-right: 6px;
  animation: pulse 2s infinite;
}

@keyframes pulse {
  0% { box-shadow: 0 0 0 0 rgba(10, 179, 156, 0.4); }
  70% { box-shadow: 0 0 0 8px rgba(10, 179, 156, 0); }
  100% { box-shadow: 0 0 0 0 rgba(10, 179, 156, 0); }
}
```
